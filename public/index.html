📖 Legado书源规则完整说明

选择器类型

  • CSS选择器 - 如 .class #id tag
  • XPath - 以 // 开头，如 //div[@class='content']
  • JSONPath - 以 $. 开头，如 $.data.list
  • 正则表达式 - 以 : 或 @regex: 开头
  • JavaScript - 以 @js: 开头

Legado特殊语法

  • class.xxx → .xxx CSS类选择器
  • id.xxx → #xxx ID选择器
  • tag.xxx → xxx 标签选择器
  • text.xxx → 包含文本xxx的元素
  • .0 .1 → 索引选择器
  • .-1 → 最后一个元素
  • @text @text() → 获取文本
  • @html @html() → 获取HTML
  • @href @src → 获取属性
  • @ownText → 获取自身文本（不含子元素）
  • ##正则 → 正则替换
  • ##正则##替换 → 正则替换为指定内容
  • && → 规则连接符
  • @@ → 规则分隔符

URL变量

  • {{key}} → 搜索关键词
  • {{page}} → 页码
