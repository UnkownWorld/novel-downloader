📖 Legado书源规则完整说明

选择器类型

  • CSS选择器 - 如 .class #id tag
  • XPath - 以 // 开头
  • JSONPath - 以 $. 开头
  • 正则表达式 - 以 : 或 @regex: 开头
  • JavaScript - 以 @js: 开头

Legado特殊语法

  • class.xxx → .xxx
  • id.xxx → #xxx
  • tag.xxx → xxx
  • .0 .1 .-1 → 索引选择器
  • @text @html @href @src → 属性提取
  • ##正则 → 正则替换
  • {{key}} {{page}} → URL变量
